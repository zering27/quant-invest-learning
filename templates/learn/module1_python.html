1.2 Pandas基础

Pandas是Python量化分析的核心库，提供了Series和DataFrame两种主要数据结构。

📊 Series数据结构

Series类似于一维数组，每个元素有一个索引标签。

代码示例：

import pandas as pd

# 创建Series
prices = pd.Series([100, 105, 102, 108, 110], 
                   index=['Mon', 'Tue', 'Wed', 'Thu', 'Fri'])
print(prices)
print(f"平均值: {prices.mean()}")
print(f"最高价: {prices.max()}")

📤 运行结果：

Mon    100
Tue    105
Wed    102
Thu    108
Fri    110
dtype: int64
平均值: 105.0
最高价: 110

📋 DataFrame数据结构

DataFrame是二维表格数据结构，类似Excel表格或SQL表，是量化分析中最常用的结构。

代码示例：创建股票数据表

import pandas as pd
import numpy as np

# 创建股票OHLCV数据
data = {
    'date': ['2024-01-01', '2024-01-02', '2024-01-03', '2024-01-04', '2024-01-05'],
    'open': [100, 102, 101, 103, 105],
    'high': [105, 107, 106, 108, 110],
    'low': [99, 100, 100, 102, 104],
    'close': [102, 105, 103, 106, 108],
    'volume': [1000000, 1200000, 1100000, 1300000, 1500000]
}

df = pd.DataFrame(data)
print(df)
print(f"\n数据形状: {df.shape}")
print(f"列名: {df.columns.tolist()}")

🧹 数据清洗

实际数据中经常有缺失值、异常值，需要进行清洗处理。

代码示例：

# 处理缺失值
df['close'].fillna(df['close'].mean(), inplace=True)  # 用均值填充
df.dropna(inplace=True)  # 删除包含缺失值的行

# 处理异常值（使用IQR方法）
Q1 = df['close'].quantile(0.25)
Q3 = df['close'].quantile(0.75)
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR
df = df[(df['close'] >= lower_bound) & (df['close'] <= upper_bound)]

# 计算日收益率
df['returns'] = df['close'].pct_change()

print(df)

1.3 NumPy数学计算

NumPy提供高性能的多维数组对象和数学函数，是Pandas和科学计算的基础。

🔢 数组运算

代码示例：

import numpy as np

# 创建数组
prices = np.array([100, 105, 102, 108, 110])

# 基本统计
print(f"均值: {np.mean(prices):.2f}")
print(f"标准差: {np.std(prices):.2f}")
print(f"最大值: {np.max(prices)}")
print(f"最小值: {np.min(prices)}")

# 计算收益率
returns = np.diff(prices) / prices[:-1]
print(f"收益率: {returns}")

# 向量化运算
ma_5 = np.convolve(prices, np.ones(5)/5, mode='valid')
print(f"5日均线: {ma_5}")

📐 矩阵运算

NumPy的矩阵运算在组合优化和风险计算中非常重要。

代码示例：

# 计算协方差矩阵
returns_matrix = np.array([[0.01, 0.02, 0.015],
                           [0.02, 0.01, 0.01],
                           [0.015, 0.01, 0.02]])

cov_matrix = np.cov(returns_matrix)
print("协方差矩阵:")
print(cov_matrix)

# 矩阵乘法
weights = np.array([0.4, 0.3, 0.3])
portfolio_return = np.dot(weights, returns_matrix.mean(axis=1))
print(f"\n组合预期收益: {portfolio_return:.4f}")

1.4 数据可视化

可视化是量化分析的重要环节，能帮助我们直观理解市场数据和策略效果。

📈 Matplotlib基础

代码示例：绘制K线图

import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

# 模拟股票数据
dates = pd.date_range('2024-01-01', periods=30)
close = 100 + np.cumsum(np.random.randn(30) * 2)

# 创建图形
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(12, 8), 
                                gridspec_kw={'height_ratios': [3, 1]})

# 绘制价格走势
ax1.plot(dates, close, 'b-', linewidth=1.5, label='收盘价')
ax1.fill_between(dates, close, alpha=0.3)
ax1.set_title('股票价格走势', fontsize=14)
ax1.set_ylabel('价格')
ax1.legend()
ax1.grid(True, alpha=0.3)

# 绘制成交量
volume = np.random.randint(1000000, 5000000, 30)
colors = ['red' if close[i] < close[i-1] else 'green' for i in range(1, 30)]
colors.insert(0, 'green')
ax2.bar(dates, volume, color=colors, alpha=0.7)
ax2.set_ylabel('成交量')
ax2.set_xlabel('日期')
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.show()
