3.1 什么是技术指标？

💡 技术指标的本质

技术指标是对价格、成交量等原始市场数据进行数学变换，得到的新数值。这些变换包括移动平均、振荡器、趋势线等。

🎯 技术指标的两大类

📈 趋势指标

用来判断市场整体趋势，如MA、MACD、ADX等。趋势指标在震荡市中容易失效。

📉 振荡指标

用来判断市场是否超买超卖，如RSI、KDJ、CCI等。振荡指标在趋势市中容易失效。

⚠️ 重要提醒

没有任何指标是万能的！不同市场环境需要使用不同的指标。量化交易的关键是建立完整的交易系统，包括选股、择时、仓位管理、风控等多个环节。

3.2 移动平均线 MA (Moving Average)

📐 原理说明

移动平均线是最基础的趋势跟踪指标。它计算一定时间周期内价格的平均值，平滑掉短期波动，显示出长期趋势。

计算公式

简单移动平均(SMA)：SMA = (C₁ + C₂ + ... + Cₙ) / n

其中 C 为收盘价，n 为周期数

⏱️ 均线周期的选择

周期 别名 用途
5日 周线 短期趋势，观察短线方向
20日 月线 中期趋势，一个交易月
60日 季线 中长期趋势，一个季度
120/250日 半年/年线 长期趋势，牛熊分界线

📊 Python实现

代码示例：计算移动平均线

import pandas as pd
import numpy as np

def calculate_ma(df, window):
    """
    计算简单移动平均线
    
    参数:
        df: 包含close列的DataFrame
        window: 移动平均周期
    
    返回:
        MA值
    """
    ma = df['close'].rolling(window=window).mean()
    return ma

# 使用Pandas内置方法（更高效）
def calculate_ma_pandas(df, periods=[5, 20, 60]):
    """计算多条均线"""
    for period in periods:
        df[f'MA{period}'] = df['close'].rolling(window=period).mean()
    return df

# 示例
data = {
    'date': pd.date_range('2024-01-01', periods=30),
    'close': 100 + np.cumsum(np.random.randn(30) * 2)
}
df = pd.DataFrame(data)
df = calculate_ma_pandas(df)

print("收盘价与均线:")
print(df[['date', 'close', 'MA5', 'MA20', 'MA60']].tail(10))

⚡ 均线的滞后性

均线是"滞后"指标，它反映的是过去的数据，对趋势变化的反应较慢。周期越长，滞后越严重。这是均线策略的固有缺陷。

3.3 MACD指标

📐 原理说明

MACD（Moving Average Convergence Divergence）称为指数平滑异同移动平均线，由Gerald Appel创建。它利用两条不同速度的EMA（指数移动平均线）来研判买卖时机。

计算公式

1. 快线EMA = EMA(close, 12)
2. 慢线EMA = EMA(close, 26)
3. DIF = 快线EMA - 慢线EMA
4. DEA/Signal = EMA(DIF, 9)
5. MACD柱 = (DIF - DEA) × 2

📊 Python实现

代码示例：计算MACD

import pandas as pd
import numpy as np

def calculate_macd(df, fast=12, slow=26, signal=9):
    """
    计算MACD指标
    
    参数:
        df: 包含close列的DataFrame
        fast: 快线周期，默认12
        slow: 慢线周期，默认26
        signal: 信号线周期，默认9
    
    返回:
        添加了DIF, DEA, MACD列的DataFrame
    """
    # 计算快线和慢线EMA
    df['EMA_fast'] = df['close'].ewm(span=fast, adjust=False).mean()
    df['EMA_slow'] = df['close'].ewm(span=slow, adjust=False).mean()
    
    # 计算DIF
    df['DIF'] = df['EMA_fast'] - df['EMA_slow']
    
    # 计算DEA（信号线）
    df['DEA'] = df['DIF'].ewm(span=signal, adjust=False).mean()
    
    # 计算MACD柱
    df['MACD'] = (df['DIF'] - df['DEA']) * 2
    
    return df

# 示例使用
data = {
    'date': pd.date_range('2024-01-01', periods=60),
    'close': 100 + np.cumsum(np.random.randn(60) * 2)
}
df = pd.DataFrame(data)
df = calculate_macd(df)

print("MACD指标:")
print(df[['date', 'close', 'DIF', 'DEA', 'MACD']].tail(10))

🎯 MACD交易信号

✅ 金叉买入

DIF从下往上穿过DEA，且都在零轴上方时，是较强的买入信号。

❌ 死叉卖出

DIF从上往下穿过DEA，是卖出信号。

3.4 RSI相对强弱指标

📐 原理说明

RSI（Relative Strength Index）由Welles Wilder创立，是一种常用的超买超卖指标。它衡量价格涨跌的幅度，用来判断市场是否过热或过冷。

计算公式

1. 上涨平均值(AA) = n日内上涨幅度的平均值
2. 下跌平均值(DA) = n日内下跌幅度的平均值（取绝对值）
3. RS = AA / DA
4. RSI = 100 - (100 / (1 + RS))

📊 Python实现

代码示例：计算RSI

import pandas as pd
import numpy as np

def calculate_rsi(df, period=14):
    """
    计算RSI指标
    
    参数:
        df: 包含close列的DataFrame
        period: RSI周期，默认14
    
    返回:
        添加了RSI列的DataFrame
    """
    # 计算价格变化
    delta = df['close'].diff()
    
    # 分离上涨和下跌
    gain = delta.where(delta > 0, 0)
    loss = -delta.where(delta < 0, 0)
    
    # 计算平均涨跌幅（使用指数移动平均）
    avg_gain = gain.ewm(com=period-1, min_periods=period).mean()
    avg_loss = loss.ewm(com=period-1, min_periods=period).mean()
    
    # 计算RS和RSI
    rs = avg_gain / avg_loss
    df['RSI'] = 100 - (100 / (1 + rs))
    
    return df

# 示例使用
data = {
    'date': pd.date_range('2024-01-01', periods=30),
    'close': 100 + np.cumsum(np.random.randn(30) * 2)
}
df = pd.DataFrame(data)
df = calculate_rsi(df, period=14)

print("RSI指标:")
print(df[['date', 'close', 'RSI']].tail(15))

...
28  2024-02-01  105.89     58.32  (RSI在0-100之间波动)

🎯 RSI交易信号

RSI值 市场状态 信号含义
> 70 超买区 可能回调，可考虑卖出
< 30 超卖区 可能反弹，可考虑买入
> 80 严重超买 强烈回调信号
< 20 严重超卖 强烈反弹信号

3.5 布林带 (Bollinger Bands)

📐 原理说明

布林带由John Bollinger发明，由三条线组成：中轨是n日移动平均线，上轨和下轨是中轨加减k倍标准差。布林带可以反映价格的波动范围。

计算公式

1. 中轨(ML) = MA(close, n)
2. 上轨(UL) = ML + k × σ
3. 下轨(DL) = ML - k × σ

通常 n=20, k=2

📊 Python实现

代码示例：计算布林带

import pandas as pd
import numpy as np

def calculate_bollinger_bands(df, period=20, std_dev=2):
    """
    计算布林带
    
    参数:
        df: 包含close列的DataFrame
        period: 周期，默认20
        std_dev: 标准差倍数，默认2
    
    返回:
        添加了BB_middle, BB_upper, BB_lower列的DataFrame
    """
    # 中轨 = 简单移动平均
    df['BB_middle'] = df['close'].rolling(window=period).mean()
    
    # 标准差
    rolling_std = df['close'].rolling(window=period).std()
    
    # 上轨和下轨
    df['BB_upper'] = df['BB_middle'] + (rolling_std * std_dev)
    df['BB_lower'] = df['BB_middle'] - (rolling_std * std_dev)
    
    return df

# 示例使用
data = {
    'date': pd.date_range('2024-01-01', periods=30),
    'close': 100 + np.cumsum(np.random.randn(30) * 2)
}
df = pd.DataFrame(data)
df = calculate_bollinger_bands(df)

print("布林带指标:")
print(df[['date', 'close', 'BB_upper', 'BB_middle', 'BB_lower']].tail(10))

🎯 布林带交易信号

📈 突破上轨

价格突破布林带上轨，可能预示强势，可考虑买入。

📉 跌破下轨

价格跌破布林带下轨，可能预示弱势，可考虑卖出。

布林带收窄（带宽变小）通常预示着即将大幅波动。
