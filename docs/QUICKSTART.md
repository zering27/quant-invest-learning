# 快速开始指南

## 环境要求

- Python 3.8+
- pip 包管理器

## 第一步：安装依赖

```bash
cd quant-invest-learning
pip install -r requirements.txt
```

主要依赖包括：
- Flask >= 2.0
- pandas >= 1.3
- numpy >= 1.21
- baostock >= 0.8.8
- plotly >= 5.0

## 第二步：启动应用

```bash
python app.py
```

如果一切正常，你将看到：

```
* Running on http://localhost:5000
* Debug mode: on
```

## 第三步：访问应用

打开浏览器，访问 http://localhost:5000

## 快速学习路径

### 1. 首页浏览 (2分钟)

访问首页，了解系统功能模块。

### 2. 学习中心 (30分钟)

访问 `/learn` 开始学习：

1. **模块1 - Python基础** (5分钟)
   - 了解Pandas数据结构
   - 理解NumPy数组运算

2. **模块2 - 数据获取** (5分钟)
   - 学习baostock数据接口
   - 了解数据预处理

3. **模块3 - 技术指标** (10分钟)
   - MA、MACD、RSI、布林带原理
   - Python实现代码

4. **模块4 - 回测框架** (5分钟)
   - 事件驱动回测原理
   - 绩效评估指标

5. **模块5 - 风险管理** (5分钟)
   - 止损止盈
   - 仓位管理

### 3. 回测实践 (10分钟)

访问 `/backtest` 进行回测：

1. 选择股票代码（如 600000）
2. 设置回测时间范围
3. 选择策略类型
4. 点击"开始回测"
5. 查看结果

## 常见问题

### Q: 回测结果在哪里查看？

A: 在回测页面点击"开始回测"后，结果会显示在页面下方，包括：
- 收益率
- 夏普比率
- 最大回撤
- 资金曲线图
- 交易记录

### Q: 支持哪些策略？

A: 当前支持：
- 均线交叉策略（MA Cross）
- RSI超买超卖策略
- 布林带策略

### Q: 如何修改策略参数？

A: 在回测页面的参数设置区域，可以修改：
- 短期均线周期
- 长期均线周期
- RSI参数
- 布林带参数

### Q: 数据来源是哪里？

A: 使用baostock免费数据源，无需注册即可使用。
